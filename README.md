## Run Locally


Clone the project


```bash

  git clone https://github.com/mostafakamal22/MERN-login-JWT

```


Go to the project directory


```bash

  cd MERN-login-JWT

```


Install dependencies


```bash

  npm install

```


Start the server


```bash

  npm run start

```
